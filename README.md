# HOOKS-TS-TEST

这是一个使用 React Hooks 开发的前端项目，  
使用了 TypeScript，  
添加了单元测试和自动化测试。

这个项目没有业务复杂度，单纯为了练手技术。

## 提前准备

1. [nodeJS](https://nodejs.org/zh-cn/) 8.0 以上版本

## 启动数据 mock 服务本地服务

1. `npm run start-server`

## 启动应用

1. `npm start`
2. 浏览器就会自动打开：`http://localhost:9000`

## function list

- react + hooks
- webpack
- local mock
- eslint
- stylelint + stylelint-order
- typescript
- jest + enzyme
